API Setup Guide

Overview

To use AI Prompt Enhancer, you need an API key from one of the supported AI providers. This guide walks you through getting API keys for each provider and configuring them in the extension.

💡 Which Provider Should I Choose?

For beginners: We recommend OpenRouter as it provides access to multiple AI models through a single API key and often has competitive pricing.

For specific needs: Choose the provider that hosts the AI model you prefer (e.g., OpenAI for GPT-4, Anthropic for Claude).

OpenRouter (Recommended)

OpenRouter provides access to multiple AI models through a single API, making it the most flexible choice for beginners.

1

Create Account

Visit openrouter.ai and click "Sign Up" to create a free account.

2

Add Credits

Go to the "Credits" section and add at least $5 to your account. New users often receive bonus credits.

3

Generate API Key

Navigate to the "Keys" section and click "Create Key". Give it a descriptive name like "AI Prompt Enhancer".

4

Copy Your Key

Copy the generated API key. It should start with "sk-or-". Keep this secure!

🎯 Recommended Models on OpenRouter

  • Claude 3 Haiku: Fast and cost-effective for most tasks
  • GPT-4o Mini: Good balance of performance and cost
  • Claude 3 Sonnet: High quality for complex prompts
  • Gemini Flash 1.5: Great for general use

OpenAI

Direct access to OpenAI's GPT models with the latest features and updates.

1

Create Account

Go to platform.openai.com and sign up for an account.

2

Verify Phone Number

Complete phone verification to access the API. This is required for all new accounts.

3

Add Payment Method

Go to "Billing" and add a payment method. You'll receive $5 in free credits to start.

4

Create API Key

In "API Keys", click "Create new secret key". Name it "AI Prompt Enhancer" and copy the key.

💰 Cost Management Tips

Set usage limits in your OpenAI account to prevent unexpected charges. GPT-4o Mini is the most cost-effective option for prompt enhancement.

Anthropic

Access to Claude models, known for their helpfulness and safety features.

1

Request Access

Visit console.anthropic.com and request API access. This may require a waitlist.

2

Complete Verification

Once approved, complete account verification including phone number confirmation.

3

Add Credits

Add credits to your account. New users typically receive $5 in free credits.

4

Generate Key

Create an API key in the dashboard. It will start with "sk-ant-".

Groq

Ultra-fast inference with open-source models, perfect for rapid prompt enhancement.

1

Sign Up

Go to groq.com and create a free account.

2

Access Console

After verification, access the GroqCloud console to manage your API keys.

3

Create API Key

Generate a new API key for use with the extension. Keys start with "gsk_".

⚡ Why Choose Groq?

  • Extremely fast response times (often under 1 second)
  • Generous free tier with rate limits
  • Open-source models like Llama and Mixtral
  • Great for real-time prompt enhancement

Perplexity

AI models with web search capabilities for up-to-date information in enhanced prompts.

1

Create Account

Visit perplexity.ai and sign up for an account.

2

Subscribe to Pro

API access requires a Pro subscription. This includes API credits and advanced features.

3

Generate API Key

In your account settings, find the API section and generate your key.

Configuring the Extension

Once you have your API key, here's how to configure the AI Prompt Enhancer extension:

1

Open Extension

Click the AI Prompt Enhancer icon in your browser toolbar, or use the keyboard shortcut.

2

Enter API Key

In the "API Configuration" section, paste your API key into the designated field.

3

Select Provider

Choose your AI provider from the dropdown menu (OpenRouter, OpenAI, etc.).

4

Choose Model

Select your preferred AI model from the available options for your provider.

5

Test Configuration

Use the "Test Enhancement" feature to verify your setup is working correctly.

🔒 Security Note

Your API key is stored locally in your browser and never sent to our servers. The extension communicates directly with your chosen AI provider.

Common Issues

❌ "Invalid API Key" Error

Solution: Double-check that you've copied the entire API key without extra spaces. Ensure your account has sufficient credits.

⚠️ "Rate Limit Exceeded"

Solution: You've hit your provider's rate limits. Wait a few minutes or upgrade your plan for higher limits.

💰 "Insufficient Credits"

Solution: Add more credits to your AI provider account. Check your billing dashboard for current balance.

🌐 "Connection Error"

Solution: Check your internet connection. Some corporate networks may block AI provider APIs.
