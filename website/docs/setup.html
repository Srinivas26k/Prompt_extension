Getting Started

Installation

Step 1: Download the Extension

Download the latest version of AI Prompt Enhancer from our website or GitHub repository.

Step 2: Install in Chrome/Edge

  1. Open Chrome and navigate to chrome://extensions/
  2. Enable "Developer mode" in the top right corner
  3. Click "Load unpacked" and select the extension folder
  4. The extension icon should appear in your toolbar

Step 3: Pin the Extension

Click the puzzle piece icon in your browser toolbar and pin AI Prompt Enhancer for easy access.

First Use

Once installed, you can start using AI Prompt Enhancer immediately:

  • Click the extension icon to open the popup interface
  • Enter your simple prompt in the text area
  • Click "Enhance Prompt" to transform it
  • Copy the enhanced prompt and use it with any AI service
