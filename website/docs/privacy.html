How We Use Information

Local Storage

Settings and preferences are stored locally in your browser to:

  • Remember your enhancement preferences
  • Maintain sidebar position and size
  • Store your preferred enhancement templates

Analytics (Optional)

If you opt-in to analytics, we collect anonymous data to:

  • Understand which features are most useful
  • Identify and fix common issues
  • Improve the user experience

Data Protection

Local Processing

All prompt enhancement happens locally in your browser. Your text never leaves your device unless you choose to copy and paste it elsewhere.

No External Servers

The core functionality of AI Prompt Enhancer works entirely offline. No data is transmitted to external servers for processing.

Secure Storage

Any data stored locally uses browser-standard security measures and is only accessible by the extension itself.

Third-Party Services

No Data Sharing

We do not share, sell, or transfer your data to any third parties. Period.

AI Service Integration

When you copy enhanced prompts to use with AI services (ChatGPT, Claude, etc.), that interaction is between you and those services directly. We have no involvement in or access to those conversations.

Browser APIs

The extension uses standard browser APIs for functionality like:

  • Storage API for saving preferences
  • Context Menu API for right-click options
  • Tabs API for sidebar functionality

Your Control

Data Management

  • Clear Data: Remove all stored preferences through browser extension settings
  • Disable Analytics: Opt-out of usage analytics at any time
  • Uninstall: Completely remove the extension and all associated data

Permissions

The extension requests minimal permissions:

  • Active Tab: To inject the sidebar on the current page
  • Context Menus: To add right-click options
  • Storage: To save your preferences locally

Children's Privacy

AI Prompt Enhancer is suitable for users of all ages. We do not knowingly collect any personal information from anyone, including children under 13.

Policy Updates

We may update this privacy policy from time to time. Any changes will be:

  • Posted on this page with an updated date
  • Announced through the extension if significant
  • Made with continued commitment to your privacy

Contact Us

If you have any questions about this privacy policy or our data practices:

  • Create an issue on our GitHub repository
