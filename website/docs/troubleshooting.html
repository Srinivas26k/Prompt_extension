Browser-Specific Issues

Chrome Issues

  • Manifest V3 Warnings: These are normal and don't affect functionality
  • Service Worker Errors: Check background.js for console logs
  • CORS Issues: Some websites may block cross-origin requests

Edge Issues

  • Extension Store: Load unpacked from chrome://extensions/ (same as Chrome)
  • Security Settings: Ensure developer features are enabled
  • Compatibility: All Chrome extension features should work

Debugging Steps

1. Check Extension Status

  1. Go to chrome://extensions/
  2. Find AI Prompt Enhancer in the list
  3. Ensure it's enabled (toggle switch is on)
  4. Click "Details" to view more information

2. View Console Logs

  1. Right-click on the extension icon
  2. Select "Inspect popup" (for popup issues)
  3. Check the Console tab for error messages
  4. For background script errors, click "Inspect views: background page"

3. Clear Extension Data

  1. Go to extension details page
  2. Click "Remove" to uninstall
  3. Restart your browser
  4. Reinstall the extension using "Load unpacked"
