Quick Start Guide

  1. Install the Extension

    Download from the Chrome Web Store or Firefox Add-ons marketplace.

  2. Get an API Key

    Sign up with your preferred AI provider (we recommend OpenRouter for beginners).

  3. Configure Settings

    Open the extension sidebar, enter your API key, and customize your preferences.

  4. Start Enhancing

    Visit any webpage with text inputs and use the extension to enhance your prompts.
